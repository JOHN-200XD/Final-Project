บริษัท เเนสเกม จำกัด (มหาชน)

บริษัท เนสเกม ขอรับประกันด้านความปลอดภัยในการซื้อสินค้ากับเรา หากลูกค้ามีปัญหาในการใช้ Key ท่านสามารถกลับไปที่หน้า วิธีซื้อ/ใช้งาน ในหัวข้อการนำ Key ที่ได้ไปใช้งาน หากลูกค้าเข้าไปดูเเล้วยังไม่เข้าใจ สามารถติดต่อกับทางเราได้ตามทางช่อง ติดต่อ หากลูกค้าต้องการ Refund สินค้า ทางเราจะดำเนินการคืนเงินให้ลูกค้าไม่เกิน 2 ชั่วโมง ถ้าหากเกินเวลาที่ได้กำหนดไว้ ให้ติดต่อทางเราทันที

เงื่อนไขในการคืนเงิน :

ประเภทเกมส์ :

-ซื้อสินค้าเเล้วยังไม่ได้ใช้งาน Key
-ใช้งาน Key ไปเเล้วไม่เกิน 2 ชั่วโมง
-ถ้าหากมีการดัดแปลงไฟล์ใน Game ผ่าน Key มาทำการ Refund ทางบริษัทจะไม่รับผิดชอบใดๆทุกกรณี

ประเภทค่าเงิน :

-ซื้อสินค้าเเล้วยังไม่ได้ใช้งาน Key
-ใช้งาน Key ไปเเล้วไม่เกิน 1 ชั่วโมง
-ถ้าหากนำไปซื้อ หรือใช้บริการ ทางบริษัทจะไม่รับ Refund เนื่องจากลูกค้าได้ใช้ในตัวสินค้าของเราไปเเล้ว

ประเภทสิทธิพิเศษ :

-ซื้อสินค้าเเล้วยังไม่ได้ใช้งาน Key
-ทางบริษัทจะไม่รับ Refund Game pass เมื่อท่านใช้งาน Key ไปเเล้วเนื่องจากลูกค้าได้ใช้ในตัวสินค้าของเราไปเเล้ว
